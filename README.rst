Webデータ収集ハンズオン
=======================

これは非ITエンジニア（Excelが使えるレベル）を対象としたハンズオン資料で、Googleスプレッドシートを使ってWebのデータを収集する方法を学習します。

:doc:`nursery`
    このハンズオンでは、Googleスプレッドシートを使って芦屋市の保育園の人気状況を可視化します。

:doc:`rakuten`
    このハンズオンでは、Google Apps Scriptを使って楽天商品検索APIを呼び出し、楽天の商品情報を収集します。
